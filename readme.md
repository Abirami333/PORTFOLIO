# Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript.

## Features

* Responsive Design
* Hero Section
* About Me Section
* Skills Section
* Projects Section
* Contact Form
* Smooth Hover Effects
* Easy Image Upload Support

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Add Images

Place your images inside the `assets` folder.

Example:

```bash id="jvnt1w"
assets/profile.jpg
assets/project1.jpg
assets/project2.jpg
assets/project3.jpg
```

Use images in HTML:

```html id="6j9gvn"
<img src="./assets/profile.jpg" alt="Profile Image">
<img src="./assets/project1.jpg" alt="Project 1">
```
